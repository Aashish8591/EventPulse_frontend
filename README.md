# 🚀 EventPulse Frontend

Modern Event Feedback Management System built using React.js and Tailwind CSS.

## ✨ Features

- Event Listing
- Search & Filter
- Submit Feedback
- View Feedback
- Responsive UI
- REST API Integration

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Axios
- React Router
- Framer Motion

## ▶️ Run

```bash
npm install
npm run dev
```
